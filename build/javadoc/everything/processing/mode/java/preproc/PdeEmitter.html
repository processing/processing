PDEEmitter: A class that can take an ANTLR Java AST and produce reasonably formatted Java code from it. To use it, create a PDEEmitter object, call setOut() if you want to print to something other than System.out, and then call print(), passing the AST. Typically, the AST node that you pass would be the root of a tree - the ROOT_ID node that represents a Java file. Modified March 2010 to support Java 5 type arguments and for loops by
